# BitNet

BitNet: 1-bit LLM architecture with ternary weights (-1, 0, +1). BitNet b1.58 adds 8-bit activations. bitnet.cpp enables fast CPU inference.

See [[summary-bitnet-cpu-inference-2024]] and [[summary-bitnet-v2-4bit-activations-hadamard-2025]].
