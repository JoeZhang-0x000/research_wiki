# Knowledge Base Index

Starter index for a markdown knowledge base.

## Structure

- `wiki/topics/` for broad overviews
- `wiki/concepts/` for atomic definitions
- `wiki/summaries/` for per-source notes

## How To Use This Starter

1. Add source files to `raw/`
2. Create summary pages for compiled sources
3. Promote recurring ideas into concept pages
4. Group related concepts and summaries under topic pages
5. Keep provenance and internal links up to date
